# Skills mobility: an open-data case study for Open Ontologies

A reproducible analysis of intergenerational **educational and skills mobility**
in England, built on the genuinely open OECD PIAAC public-use files (2012 and
2023), and a worked example of the **Open Ontologies** harmonisation layer
applied to open public-sector data.

**Headline finding:** adults' numeracy is strongly predicted by their parents'
education, and the gap persists even among people who hold the *same*
qualification. Qualifications understate the advantage of social origin.

See [case-study.md](case-study.md) for the narrative (gov.tesseract.academy
house style) and [technical-note.md](technical-note.md) for full methods.

## What is here

```
config.R                 paths, the cross-cycle variable map, constants
R/
  01_download.R          fetch the open PIAAC CSV + SPSS files (idempotent)
  02_harmonise.R         build one harmonised analysis table on a 3-band ladder
  03_analyse.R           estimates: PVs by Rubin's rules, replicate-weight SEs
  04_visualise.R         accessible figures (PNG) + per-figure CSVs
  lib_piaac.R            reusable PIAAC estimators (PVs, cycle-aware variance)
ontology/
  build_scheme.R         generate the coded SKOS variable scheme
  validate_scheme.py     load + check it in the Oxigraph engine, write coverage
  skills-mobility-scheme.ttl / .jsonld   the published scheme
  entities.csv           entity list (every concept traces to a PIAAC source)
  coverage-report.md     engine-generated provenance + coverage report
outputs/
  figures/               4 figures (PNG) and their underlying CSVs
  tables/                all estimate tables (CSV)
```

Raw data (`data/raw/`, 30-46 MB per file) is regenerated by `01_download.R` and
is not committed.

## Reproduce

Requires R (with `intsvy`, `survey`, `mitools`, `tidyverse`, `haven`, `jsonlite`)
and, for the ontology step, Python 3.11 with `pyoxigraph`.

```bash
Rscript R/01_download.R
Rscript R/02_harmonise.R
Rscript R/03_analyse.R
Rscript R/04_visualise.R
Rscript ontology/build_scheme.R
.venv/bin/python ontology/validate_scheme.py
```

## Provenance and honesty

- Data: OECD PIAAC public-use files only. No restricted or licensed microdata.
- Estimates validated against the published OECD England toplines.
- All results are descriptive associations, not causal claims.
- Every ontology concept traces to a documented PIAAC source variable and the
  official OECD value label. Coverage report records 100% provenance.

Source: OECD Survey of Adult Skills (PIAAC), England, 2012 and 2023. Analysis by
The Tesseract Academy, June 2026.
