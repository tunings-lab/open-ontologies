All are real-data-backed (292 frames from the public NCAP Air Photo Finder API,
harvested 2 Jul 2026). Draft captions below — edit to taste. No em dashes per house style.

The three-country maps are generated by `demo/world-maps.html` (Leaflet + CARTO
basemap), screenshotted with headless Chrome at `#all`, `#canada`, `#usa`:

```bash
python3 -m http.server 8799 &
CHROME="/Applications/Google Chrome.app/Contents/MacOS/Google Chrome"
for v in all:three-country canada:canada usa:usa; do
  "$CHROME" --headless=new --window-size=1680,1000 --virtual-time-budget=11000 \
    --force-device-scale-factor=2 --screenshot="assets/real-demo-${v#*:}.png" \
    "http://localhost:8799/demo/world-maps.html#${v%:*}"
done
```
